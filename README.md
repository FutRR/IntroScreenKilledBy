# IntroScreenKilledBy

This addon adds the name and faction (if NPC) of the NPC/Mutant/Anomaly that killed you on the death screen.

All credits go to Serious, 8BitCat and eslider

This addon basically merges IntroScreen and [Head, Eyes] Death Screen to add the killer name to the intro screen.

This is my first time modding Anomaly, so bugs are possible. I have not thoroughly tested it but it should work fine.

Load after the 3 requirements.

## Requirements

- [Tagged Patches](https://www.moddb.com/mods/stalker-anomaly/addons/tagged-patches) by Utjan (required by eslider's [Head, Eyes] Death Screen code)
- [Intro Screen](https://www.moddb.com/mods/stalker-anomaly/addons/intro-screen-animation) by Serious
- [IntroScreenAddon](https://discord.com/channels/912320241713958912/1361848518091870248/1517901907824214104) by 8BitCat
