# IntroScreenKilledBy

This addon adds the NPC/Mutant name that killed you on the death screen.

All credits go to Serious, 8BitCat and eslider

This addon basically merges IntroScreen and [Head, Eyes] Death Screen to add the killer name to the intro screen.

This is my first time modding Anomaly, so bugs are possible. I have not thoroughly tested it but it should work fine.

## Requirements

- [Tagged Patches](https://www.moddb.com/mods/stalker-anomaly/addons/tagged-patches) by Serious
- [Intro Screen Animation](https://www.moddb.com/mods/stalker-anomaly/addons/intro-screen-animation) by 8BitCat
